# Keymap repository

mac os x版的 Settings Repository，由于mac os x与windows系统差别过大，设置文件难以通用，故新建此仓库。
使用方式参考:[settings-repository(windows版)](https://github.com/vegetayu/settings-repository)

# 快捷键说明

- 弹窗交互：查找，重写，
